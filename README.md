## ğŸ›† Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/rajparkhi/apple_website.git
   ```

2. **Navigate to the project directory**

   ```bash
   cd apple_website
   ```

3. **Install dependencies**

   ```bash
   npm install
   ```

   or

   ```bash
   yarn install
   ```

4. **Run the development server**

   ```bash
   npm run dev
   ```

   or

   ```bash
   yarn dev
   ```

5. **Open your browser and visit**
   ```
   http://localhost:5173
   ```
